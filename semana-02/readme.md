# GIT
1. Iniciamos un repositorio de GIT git init
2. Configuramos el usuario 
- git config user.name "Lucila Corvalan"
- git config user.email lucila.corvalan@
3. Agregamos los archivos al repositorio git add README.md
4. Hacemos el commit(foto) git commit -m "Proyecto inicial"
5. Vinculamos el repositorio remoto   git remote add origin https://github.com/lucila-corvalan06/pwa.git
6. Realizamos el push git push origin master
